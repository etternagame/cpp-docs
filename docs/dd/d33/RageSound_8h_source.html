100  /* If bPrecache == true, we'll preload the entire file into memory if
101  * small enough. If this is done, a large number of copies of the sound
102  * can be played without much performance penalty. This is useful for
103  * efficiently playing keyed sounds, and for rapidly-repeating sound
104  * effects, such as the music wheel.
105  *
106  * If cache == false, we'll always stream the sound on demand, which
107  * makes loads much faster.
108  *
109  * If the file failed to load, false is returned, Error() is set
