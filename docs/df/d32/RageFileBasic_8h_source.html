130  /*
131  * If read buffering is enabled, m_pReadBuffer is the buffer, m_pReadBuf is
132  * the current read position in the buffer, and m_iReadBufAvail is the
133  * number of bytes at m_pReadBuf. Note that read buffering is only enabled
134  * if:
135  *
136  * - GetLine() is called (which requires buffering to efficiently search
137  * for newlines);
138  * - or EnableReadBuffering() is called
139  *
140  * Once buffering is enabled, it stays enabled for the life of the object.
141  *
142  * If buffering is not enabled, this buffer will not be allocated, keeping
143  * the size overhead of each file down. Layered RageFileBasic
144  * implementations, which read from other RageFileBasics, should generally
