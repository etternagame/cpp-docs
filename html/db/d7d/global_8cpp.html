void NORETURN sm_crash ( const char *  reason = "Internal error")

A crash has occurred, and we're not getting out of it easily.

For most users, this will result in a crashinfo.txt file being generated. For anyone that is using a debug build, a debug break will be thrown to allow viewing the current process.
